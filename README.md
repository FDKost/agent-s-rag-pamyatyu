# Агент с RAG-памятью

## Requirements
- [high] Подключение к Qdrant: Добавить импорт qdrant_client, настроить клиент, создать коллекцию и загрузить векторные представления документов.
- [high] Интеграция Ollama: Импортировать Ollama из langchain_community, настроить модель, убедиться в работе генерации ответов.
- [high] Построение RAG‑pipeline: Собрать цепочку LangChain, включающую RetrievalQA, использовать Qdrant как векторный хранилище и Ollama как LLM, добавить обработку пользовательских запросов.
- [normal] Тесты и пример: Написать unit‑тесты для проверки корректности работы RAG‑pipeline и предоставить README‑пример использования.
